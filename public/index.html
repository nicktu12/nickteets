Statify

Statify is an application built to display a Spotify user's listening statistics on an interactive dashboard. Using the Spotify API, users are able to see different data sets over different periods of time. Statify includes a backend application built in Node and Express for communicating to the Spotify database while hiding sensitive information from the front-end application.


React.js, Redux.js, React Router, Redux Sagas, Node.js, Express, SCSS

Construction Forms

Construction Forms aims to eliminate the inefficiencies of construction documentation at the field level. This includes a native mobile app that allows tradespeople to fill out their usual planning and safety forms on their iPhone or Android device. Most importantly, the app includes a web portal for administrators to gather all the form data and sort it by project or employee. This eliminates paper, filing time and data entry that would usually fall on the shoulders of a project admin or engineer.


React.js, React Native, Node.js, Express

Fly Over Media

Fly Over Media is a cultural education non-profit which produces, supports and publishes interactive, multimedia journalism pieces about rural and underrepresented communities. Fly Over Media began as Fly Over Me: Valentine, a crowdsourced multimedia documentary project focusing on the communities of the Nebraska Sandhills.
